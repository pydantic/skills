# pydantic

Validate and serialize data with [Pydantic](https://docs.pydantic.dev/).

## Features

- SKILL.md with core Pydantic patterns (models, constraints and field metadata, custom validators, type coercion, forward annotations, model subclasses and discriminated unions)

## Install

```bash
claude plugin marketplace add pydantic/skills
claude plugin install pydantic@pydantic-skills
```

## Codex

`pydantic` is not currently listed as a Codex plugin in the Pydantic marketplace. Codex users can still use the
standalone cross-agent skill at `skills/pydantic/`.
