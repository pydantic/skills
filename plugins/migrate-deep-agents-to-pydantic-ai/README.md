# Migrate Deep Agents to Pydantic AI

Migrate LangChain Deep Agents projects to Pydantic AI without silently changing state, persistence, delegation, approval, or sandbox semantics.

## Features

- Repository-wide inventory workflow for Deep Agents projects
- Feature-by-feature migration mappings and large-project playbooks
- Guidance for composing rich Pydantic AI primitives and optional Harness capabilities
- Clear application boundaries for durable state, artifacts, background work, memory, and remote sandboxes
- Parity validation guidance for source and target behavior

Pydantic AI Harness is an optional target capability source, not the owner of this skill.

## Install

```bash
claude plugin marketplace add pydantic/skills
claude plugin install migrate-deep-agents-to-pydantic-ai@pydantic-skills
```
